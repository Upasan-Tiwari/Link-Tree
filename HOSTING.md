# How to Host Your Link-Tree Permanently

This guide will help you host your Link-Tree website for free and use it in your social media bio.

## üöÄ Recommended Options (All Free!)

### Option 1: GitHub Pages (Recommended - Easiest)
**Best for:** If you already use GitHub  
**URL Format:** `https://yourusername.github.io/Link-Tree`

#### Steps:
1. **Push your code to GitHub:**
   ```bash
   git push origin main
   ```

2. **Enable GitHub Pages:**
   - Go to your repository on GitHub: `https://github.com/Upasan-Tiwari/Link-Tree`
   - Click **Settings** (top right)
   - Scroll down to **Pages** (left sidebar)
   - Under **Source**, select **Deploy from a branch**
   - Choose **main** branch and **/ (root)** folder
   - Click **Save**

3. **Wait 1-2 minutes**, then visit: `https://upasan-tiwari.github.io/Link-Tree`

4. **Use in your bio:** Copy the URL and paste it in your Instagram/TikTok/YouTube bio!

---

### Option 2: Netlify (Super Easy - Drag & Drop)
**Best for:** Quickest setup, no git needed  
**URL Format:** `https://your-site-name.netlify.app` or custom domain

#### Steps:
1. **Go to:** https://www.netlify.com
2. **Sign up** (free) with GitHub/Google/Email
3. **Drag & Drop Method:**
   - On Netlify dashboard, drag your entire `Link-Tree` folder
   - Wait for deployment (takes ~30 seconds)
   - Get your live URL instantly!

4. **Optional - Customize URL:**
   - Click **Site settings** ‚Üí **Change site name**
   - Change to something like `upasan-links` ‚Üí `https://upasan-links.netlify.app`

5. **Use in your bio:** Copy the Netlify URL!

---

### Option 3: Vercel (Fast & Modern)
**Best for:** Fast performance, easy updates  
**URL Format:** `https://link-tree-xyz.vercel.app`

#### Steps:
1. **Go to:** https://vercel.com
2. **Sign up** with GitHub (recommended)
3. **Click "Add New Project"**
4. **Import your GitHub repository** (Link-Tree)
5. **Click Deploy** (no configuration needed)
6. **Get your live URL** in seconds!

---

### Option 4: Cloudflare Pages (Fast & Free)
**Best for:** Best performance worldwide  
**URL Format:** `https://link-tree.pages.dev` or custom domain

#### Steps:
1. **Go to:** https://pages.cloudflare.com
2. **Sign up** (free)
3. **Connect your GitHub account**
4. **Select your Link-Tree repository**
5. **Click "Begin setup"** ‚Üí **Save and Deploy**
6. **Get your live URL!**
